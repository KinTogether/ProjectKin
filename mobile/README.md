# Mobile (Flutter)

This folder will contain the Flutter app. It is initialized as part of [issue #12](https://github.com/KinTogether/ProjectKin/issues/12).

Planned structure:

```
mobile/
  lib/
    core/         # shared utilities, constants, theme
    data/         # repositories, data sources, models
    domain/       # business logic, use cases
    presentation/ # screens, widgets, state management
  test/
  integration_test/
  pubspec.yaml
  analysis_options.yaml
```
